Shell运算符与条件判断

Shell运算符与条件判断

运算符

  1. 基本语法
    • “$((运算式))”或“$[运算式]” 【推荐】
    • expr m + n 注意expr运算符间要有空格
    • expr *, /, % 乘，除，取余

条件判断

  1. 基本语法

    [ condition ] (注意condition前后要有空格)

  2. 示例

    [ atguigu ] 返回true

    [] 返回false

    [condition] && echo OK || echo notok 条件满足，执行后面的语句

  3. 常用判断条件（仅仅是判断）

    • 整数的比较

      -lt 小于

      -le 小于等于

      -eq 等于

      -gt 大于

      -ge 大于等于

      -ne 不等于

    • 字符串 =

    • 文件权限进行判断

      -r 有读的权限

      -w 有写的权限

      -x 有执行的权限

    • 文件类型进行判断

      -f 文件存在并且是一个常规的文件

      -e 文件存在

      -d 文件存在并是一个目录

case语法：

case $变量名 in

"值1"）

如果变量的值等于值1，则执行程序1

;;

"值2"）

如果变量的值等于值2，则执行程序2

;;

…省略其他分支

… *）

如果变量的值都不是以上的值，则执行此程序

;;

esac

循环控制

基本语法

  1. for 变量in 值1 值2 值3…

​ do

​ 程序

​ done

  1. for (( 初始值;循环控制条件;变量变化))

    ​ do

    ​ 程序

    ​ done

示例：

for i in $*           # 这边不使用双引号的时候 $*和$@的效果是一样的  有几个参数就遍历几次
  do                  # 一旦用了双引号 "$*" 则是整个参数  而"$@"则会将参数拆分开
   echo  $i
  done

1-100累加：

SUM=0
for((i=1;i<=100;i++))
  do
    SUM=$[$SUM+$i]         //注意这边的运算符
  done
echo $SUM

while [ $A -le $1 ]
 do
   SUM=$[$SUM+$A]

done
